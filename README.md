# The-financial-calculator
Программа позволяет оценить успешность финансового проекта при помощи расчета ключевых финансовых метрик ROI (return on investments), NPV (net present value), IRR (internal rate of return) и Payback Period (срок окупаемости). Расчет данных показателей поможет оценить прибыльность проекта.
